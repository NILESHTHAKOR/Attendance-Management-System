# Student Attendance System (JavaScript)

Converted from TypeScript to JavaScript.

## Getting Started

```bash
npm install
npm run dev
```

## Changes from TypeScript version

- All `.tsx` files renamed to `.jsx`
- All `.ts` files renamed to `.js`  
- TypeScript interfaces and type declarations removed
- Type annotations removed from function parameters
- Generic type parameters removed from React hooks and forwardRef calls
- `tsconfig.json` and TypeScript-specific dev dependencies removed
- `vite.config.ts` → `vite.config.js`
- `tailwind.config.ts` → `tailwind.config.js`
- `eslint.config.js` updated to use JS-specific rules
